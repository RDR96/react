# Entity Inbox Email Address

The `<EntitySelector>` component lets you select an entity from a list of entities in an entity group.

## Customization Options

| Themeable | Variations | Custom Component |
| :-------: | :--------: | :--------------: |
|    ✅     |     ❌     |        ❌        |

## Props

| Name                | Type                                              | Required | Description                                                     |
| :-------------------| :-------------------------------------------------| :--------| :---------------------------------------------------------------|
| onSelect            | `(entity?: Mercoa.EntityResponse) => void`        |          | Callback when an entity is selected, if not provided, the entity will be set at the session level |
| allowClear          | `boolean`                                         |          | If true, allows the user to clear the selection and the selected entity at the session level                 |
